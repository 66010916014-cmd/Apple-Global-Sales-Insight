# Apple Global Sales Insight

## Overview

Apple Global Sales Insight is an interactive Power BI dashboard developed to analyze Apple product sales performance using Power BI, Power Query, and DAX.

The dashboard provides business insights through KPI visualization, sales trends, product performance analysis, and revenue analysis.

---

## Technologies

Power BI
Power Query
DAX
Microsoft Excel
Star Schema
Data Modeling

---

## My Responsibilities

Cleaned and transformed sales data using Power Query.
Designed a Star Schema data model in Power BI.
Developed DAX measures for business KPIs.
Created an interactive dashboard with multiple report pages.
Visualized sales trends and product performance to support business insights.

---

## Dashboard Features

Total Revenue
Total Units Sold
YoY Growth
Profit Margin
Sales Trend Analysis
Product Performance Analysis
Interactive Dashboard
Business KPI Monitoring
